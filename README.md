# My Personal n8n Workflows

This repository stores my personal n8n automation workflows.

## How to Import

1. Open your n8n instance
2. Go to **Workflows** → **Import from File**
3. Choose any `.json` file from the `workflows/` folder

## Folder Structure

- `workflows/personal/` → All my personal automations

## Tips

- Workflows are saved as plain JSON files
- Use clear names: `daily-backup.json`, `twitter-digest.json`, etc.
- You can also use n8n's **Source Control** feature to connect this repo directly
